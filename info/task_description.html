Your goal is create the fastest sorting algorithm ever - Sort By Removing.

You just need to remove all elements from a given array that are in incorrect order. As always, we have some conditions:

  • it is asc sorting
  • first element should always stay in the result list
  • the result of empty list is empty list
  • elements are always integers

Input: Two arguments. Both are ints.

Output: Int.

Example:
